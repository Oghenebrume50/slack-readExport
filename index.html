Dev-Resources API Documentation

This will server as basic documentation of the API and detail the various endpoints available

User Endpoints

/user/auth/discord

get request to authenticate users

/user/logout

get request logs out an authenticate user

/user/profile

get Request returns user details if logged in

post Request stores users session directly after authenticating

Resource Endpoints

/resource/all

get request to list all resources

/resource/:user/bookmark

get request to return all resource the specified user has bookmarked

post Request to save a resource for the specified user

Stats Endpoints

/stats

get request placeholder

Contributor Endpoints

/contributor

get request returns a JSON list of contributions
